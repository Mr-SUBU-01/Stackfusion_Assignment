# 🚀 Stackfusion Full Stack Assignment

## 📌 Overview

This project is a full-stack parking management system built using:

* Django (Backend API)
* Go (Service layer)
* Next.js (Frontend)

The goal of this assignment was to debug and fix issues across all three services.

---

## 🛠️ Issues Identified & Fixed

### 🔹 Backend (Django)

* Fixed incorrect slot availability logic
* Resolved booking state inconsistency
* Corrected serializer configuration issues

### 🔹 Backend (Go)

* Fixed SQL query issues in active bookings report
* Ensured correct filtering and data return

### 🔹 Frontend (Next.js)

* Fixed API integration issues with backend
* Improved slot refresh after booking
* Handled loading and error states

---

## ⚙️ How to Run

### 1️⃣ Backend (Django)

```bash
cd backend-django
python manage.py runserver
```

### 2️⃣ Backend (Go)

```bash
cd backend-go
go run main.go
```

### 3️⃣ Frontend

```bash
cd frontend-nextjs
npm install
npm run dev
```
